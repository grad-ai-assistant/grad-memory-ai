# Grad Memory AI — Product Vision

## Overview

Grad Memory AI is an AI-powered personal academic memory assistant designed to help graduate school applicants retrieve precise information from their academic and professional documents.

Graduate applications often require applicants to repeatedly look up specific details such as grades, course names, university addresses, research topics, and recommender information. This process is time-consuming and error-prone.

Grad Memory AI aims to solve this by acting as a persistent, intelligent memory system that can instantly retrieve exact factual information from uploaded documents.

---

## Problem Statement

Graduate applicants face the following challenges:

- Repeatedly searching through transcripts, SOPs, and CVs
- Difficulty recalling precise academic details
- Risk of inconsistencies across applications
- Manual summarisation of experiences
- Lack of structured academic memory

Existing “chat with PDF” tools are not designed for high-precision academic retrieval.

---

## Proposed Solution

Grad Memory AI provides:

- Persistent document memory (upload once, use anytime)
- Precise factual retrieval using Retrieval-Augmented Generation (RAG)
- Numeric extraction for grades, CGPA, scores
- Structured one-line form-ready answers
- Source citation for verification
- Support for multiple SOP and LOR versions

---

## Target Users

- Graduate school applicants
- Scholarship applicants
- Academic researchers
- Early career professionals

---

## Phase 1 Scope

- Single-user personal assistant
- Document upload and storage
- RAG-based precise Q&A
- Source attribution
- Basic web chat interface

---

## Future Scope

- Multi-user SaaS platform
- Academic timeline visualisation
- Auto form filling integrations
- Application tracking
- Document intelligence analytics

---

## Vision

To build a structured academic intelligence system that transforms how students manage and utilise their academic history.
